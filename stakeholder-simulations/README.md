# Stakeholder Simulations

43 realistic stakeholder scenarios designed to build the skills CISOs use in the hardest conversations. Each simulation is immediately runnable in the CISO Co-Pilot project.

## Difficulty ratings

- **1-2:** Manageable conversations. Good for building foundational skills and confidence.
- **3:** Genuinely hard. Requires preparation, composure, and specific tactical skills.
- **4:** Politically or legally complex. Multiple stakeholders, competing interests, real consequences.
- **5:** Career-defining moments. Incomplete information, time pressure, no clear right answer.

## How to use

1. Pick a simulation that matches a conversation you expect to have or want to practice
2. Read the full scenario file to understand the context, pressure points, and recommended approach
3. Copy the **Activation Prompt** into your CISO Co-Pilot project
4. Have the conversation. Push back. Get pushed back on.
5. When done, copy the **Debrief Prompt** into the same conversation
6. Review your performance scores and identify what to work on

## File structure

Every simulation follows an identical structure:

- **Scenario Brief:** Who, what, and why this conversation is happening
- **What Makes This Hard:** The specific challenges that make this scenario difficult
- **What They Will Not Say Out Loud:** The subtext driving the stakeholder's behavior
- **Pressure Points and Tactics:** Exact phrases and behaviors the stakeholder will use
- **How to Handle It:** Tactical moves for the CISO
- **Activation Prompt:** Paste this into CISO Co-Pilot to start the simulation
- **Debrief Prompt:** Paste this after the simulation to extract lessons and scores
