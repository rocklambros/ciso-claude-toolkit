# Presentation

This directory contains the speaker notes for the CISO AI toolkit talk. The slide deck itself lives in Gamma and is not stored in this repository. These speaker notes are what Rock says aloud during each slide.

Any CISO can use these materials to deliver their own version of the talk. The speaker notes are written in Rock's voice but contain the substance, not the personality. Adapt the tone to match your own speaking style. Keep the structure.

## Directory structure

```
presentation/
└── speaker-notes/
    ├── slide-03-the-actual-number.md
    ├── slide-04-where-the-number-comes-from.md
    ├── slide-05-the-math-you-avoided.md
    └── slide-08-what-youre-about-to-watch.md
```

## How to use these notes

Each file contains the full spoken content for one slide. Read them as scripts you would say to a room, not as bullet points you would put on a screen. The slides themselves should be minimal: a number, a formula, a diagram. The substance comes from what you say.

If you are building your own Gamma deck:

1. Create slides with minimal visual content (the number, the formula, the key phrase)
2. Put these speaker notes in the Gamma presenter notes field for each slide
3. Practice the transition from Slide 8 into the live demo at least twice, because that is where the format shifts from presentation to performance

## Adapting for your audience

The math slides (3, 4, 5) assume the audience includes CISOs who have been told AI is dangerous but have not seen anyone quantify the comparison to existing SaaS tools. If your audience already accepts that AI risk is comparable to existing tool risk, shorten these slides and spend more time on the demo.

Slide 8 is the bridge between slides and demo. If you are not running the live demo, this slide does not work. Either commit to the live demo or replace this slide with a recorded walkthrough.
