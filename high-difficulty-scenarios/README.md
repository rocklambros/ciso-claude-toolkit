# High-Difficulty Scenarios

7 crisis decision-making scenarios at difficulty 4 and 5. These are not stakeholder simulations. The pressure is situational, not conversational. The CISO must make real-time decisions with incomplete information, time constraints, and consequences that extend beyond the immediate moment.

## How to use

Each scenario includes a prompt you can paste into CISO Co-Pilot. The prompt provides the scenario context and asks the model to help you think through each decision point with options, risks, and a recommendation.

These scenarios work best in SUPPORT mode for real-time decision support or in TRAIN mode for practice.

## File structure

Every scenario follows an identical structure:

- **What Just Happened:** The situation as it arrives, incomplete and ambiguous
- **What You Know:** Confirmed facts only
- **What You Do Not Know:** The critical unknowns
- **Why This Is Harder Than It Looks:** Hidden complexity and second-order effects
- **The Decision Points:** Specific decisions in priority order
- **How to Run This in CISO Co-Pilot:** The exact prompt to paste
- **What Good Looks Like:** The outcome a strong CISO produces
