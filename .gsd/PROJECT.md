# ciso-claude-toolkit

## Overview
A toolkit of Claude system prompts designed for cybersecurity leadership, CISO advisory, and prompt engineering. Contains curated, production-ready system prompts that can be used with Claude to support strategic security work.

## Project Type
Prompt/content project (no application runtime)

## Structure
```
system-prompts/          # Claude system prompts
├── ciso_copilot_system_prompt.md    # CISO strategic advisor prompt
└── prompt_engineer_system_prompt.md # Prompt optimization prompt
```

## Key Components

### CISO Copilot System Prompt
- Four-pillar framework: Business, Communication, Leadership, AI/Agentic Security
- Three operating modes: TRAIN, SUPPORT, SPAR
- Stakeholder simulation (CFO, CEO, Board, GC, CIO/CTO, etc.)
- Framework coverage: NIST CSF, ISO 27001, FAIR, COSO ERM, OWASP

### Prompt Engineer System Prompt
- Claude 4 prompt optimization aligned with Anthropic best practices
- Analysis framework with 10 evaluation criteria
- Reusable XML blocks for common patterns
- Context lifecycle management for long interactions
